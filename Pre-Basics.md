# Pre-Basics for CS

## What is Computer:-
- It is an `electronic device` that can `process & manipulate data`.

- Computers can't do anything without a program that's why sometimes they are called as `Dumb Machines`.

- We use computers because they process data quickly, accurately, and efficiently, automate tasks faster than humans.


## What is Computer Science:-
- In layman's term, It is study of data, data processing, data transformation & data presentation.

- Computer science is the study of how computers process, store, and analyze data, as well as how to design software and systems to solve problems.


## Programming Language:-
- It is a ```way to communicate with the machines/computers```.

- It is used to write programs and give instructions to computers.


## Program:-
- It is ```set of instructions``` that a machine follows to perfrom a task.


## Algorithm:-
- It is step by step set of instructions used to solve a given problem.


## Flowchart:-
- It is pictorial representation of an algorithm.


## Data Structure:-
- It is a way of Storing & Organizing the data in efficient manner, so that we can easily & effciently handle them.


## Debugging:-
- It is ```process of finding the bugs/errors & remove/fix them```.


## Compiler:-
- It translates source code into machine code.

- It ```execute the whole program at once before the program runs/execution```.

- Examples:-
    1. C#
    1. Java
    1. C & C++


## Interpreter:-
- It translates source code into machine code.

- It ```execute the program line by line during execution```.

- Examples:-
    1. Javascript
    1. Python


## Statements:-
- It is single fully executable instruction/line of code.

```cs
int age = 21; // statement

Console.WriteLine(age); // statement
```


## Expression:-
- It is combination of operands & operators.

- It usually produce a single value/result.

```cs
10 + 20; // expression
 
age > 18; // expression

num1 + num2; // expression
```
