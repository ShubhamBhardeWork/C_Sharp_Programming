# Math Class

## Math Class:-
- `Math` class is a built-in class in C#.

- It is used to perform mathematical operations.

- All methods of `Math` class are `static`.

- Most methods return `double` type values.

- It is available in `System` namespace.


## Math Methods:-
1. Math.PI
1. Math.Round()
1. Math.Floor()
1. Math.Ceiling()
1. Math.Truncate()
1. Math.Abs()
1. Math.Pow()
1. Math.Sqrt()
1. Math.Max()
1. Math.Min()


## Hands-on:-
```cs
Console.WriteLine(Math.PI); // 3.14159265358979

Console.WriteLine(Math.Round(10.6)); // 11

Console.WriteLine(Math.Floor(10.9)); // 10

Console.WriteLine(Math.Ceiling(10.1)); // 11

Console.WriteLine(Math.Truncate(10.9)); // 10

Console.WriteLine(Math.Abs(-20)); // 20

Console.WriteLine(Math.Pow(2, 3)); // 2*2*2 = 8

Console.WriteLine(Math.Sqrt(25)); // 5

Console.WriteLine(Math.Max(10, 20)); // 20

Console.WriteLine(Math.Min(10, 20)); // 10
```
